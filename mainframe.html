Sevlet JDBC Procedure


         Procedure:

               1.Create a database in mysql using create database ;

               2.Use the database to create tables under it, using use 

               3.Create an employee table with fields like emp_id, ename,DOB, phone_no, salary etc.

               4.Create java file that will contain all database operation on a database.

               5.Connect the java program with the database using JDBC.

               6.Employee table is connected to the program by its database name.

               7.The connection is created using Java Database connectivity.

               8.The Query operations like Register(insert),View(select),Update,Delete are performed on the database by extending java class.

               9.The recordset and connection is closed finally.
